# 🍣 Sushi Restaurant Website

A responsive, modern landing page for an authentic Japanese restaurant.
Built with vanilla HTML, CSS, and JavaScript — no frameworks required.

## 🔗 Live Demo

👉 [View Demo](https://engnada-s.github.io/Sushi-Restaurant-Website/)

## 🛠️ Tech Stack

- **HTML5** — semantic structure
- **CSS3** — custom properties, grid, flexbox
- **JavaScript (ES6)** — scroll behavior, active nav, menu toggle
- **Swiper.js** — menu tabs & slider
- **ScrollReveal** — entrance animations
- **Remix Icons** — icon set
- **Google Fonts** — Lora + Montserrat

## 🚀 Getting Started

No build tools needed. Just open `index.html` in your browser.
```bash
git clone https://github.com/EngNada-S/Sushi-Restaurant-Website.git
cd Sushi-Restaurant-Website
open index.html
```
